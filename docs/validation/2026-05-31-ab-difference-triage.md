## 패턴별 로직 반영 과제 (B)

1. **처분 cash-flow bridge 일반화**: `처분 장부금액 − 처분손익(− 감가상각누계 등) = 처분대가`. 현재 일부만 적용. 현대건설·현대모비스·지누스·한화시스템·더존·GS리테일 PPE/무형 처분.
2. **CFS 합산 라인 분해**: `유형자산 및 사용권자산의 취득`처럼 PPE+ROU 합산 표시 → 주석 PPE취득 + ROU취득 대사. 롯데하이마트.
3. **취득 미지급/건설중 bridge**: 주석 취득 > CFS → 미지급 취득 증가 차감 / 건설중자산. 현대차(350B)·해성디에스(17B)·더존(1.2B)·GS리테일(8.4B)·세아제강(5.65B).
4. **financing 구성요소 완전성·분류**: 풍산 차입 +100.0B / 사채 −100.0B 거의 상쇄 → 유동성장기차입↔사채 분류 swap. 현대건설·해성디에스 차입 component 누락.
5. **balance 오선택 방어**: 롯데하이마트 무형 balance diff 572.9B (자산 총액 초과) → 명백한 wrong-row 매칭. 차이가 계정 규모를 초과하면 매칭 거부 + parse_uncertain 강등.
6. **balance 영업권/정부보조금 조정**: 현대건설 무형(334B)·풍산 PPE(−7.83B).
